## Structure

```
ProjectDrive/
  Group-X/
    <Project>/
      Data Cleaning/   README.md
      Data Analysis/   README.md
      Dashboard/       README.md
      Report/          README.md
```

## Naming notes
- `A-B Test` is used instead of `A/B Test` because a slash is a path separator and would break the folder.
- Each project is owned individually by one member, who is responsible for all four role folders
  (Data Cleaning, Data Analysis, Dashboard, Report) for their own project.
